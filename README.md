# Block theme for the Wicked Evolutions ecosystem. Native Gutenberg, dark/light mode, documentation + editorial.

## Requirements

- WordPress 6.7+
- PHP 8.0+

## Features

- Self-hosted variable fonts (Syne, JetBrains Mono) — no Google Fonts requests
- Dark background, light text default
- Category-based single post template resolution
- Auto-generated table of contents from headings
- Responsive, constrained content width (800px / 1200px wide)
